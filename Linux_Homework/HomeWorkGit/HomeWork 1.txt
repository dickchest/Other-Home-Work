 
1. Как скопировать на свой компьютер репозиторий из github.com, что должно быть сделано предварительно?
команда git clone <SSH adress>
предварительно нужно скопировать SSH адрес из Code - Local - SSH 

2. Как создать git репозиторий из локального каталога на своем компьютере. Как его отправить на github.com?
git init - в папке с контектом
git add .
git commit -m "initialisation"
создаем новый репозиторий на ГитХабе
git remote add origin git@github.com:<Имя пользователя/имя репозитория>
git branch -M main = создание главной ветки
git push -u origin main = -u - создание ветки одновременно с пушем

3. Как получить на локальный компьютер изменения с github.com
git pull

4. Какую информацию нам дают команда :
git status = текущий статус файлов и их состояние
git log = информация об истории коммитов
git remote -v 
Просмотр списка настроенных удаленных репозиториев. -v - просмотр адресов для чтения и записи привязанных к репозиторию.
