1. Как отказаться от изменений в файле (файлах) находящийся в рабочем каталоге (work directory)?
git restore <file>

2. Как вернуть файл из индекса в рабочий каталог?
git restore --staged <file>

3. Как отменить ошибочный commit ?
git commit –amend
добавление и редакторование коммита

git reset , ^[^^^^] - на сколько шагов или ~[1]

4. Как создать новую ветку и переключится в нее?
git checkout -b <name>

5. В чем отличение между fast-forward merge  and non fast-forfard merge?
fast-forward - git автоматически соединяет ветки, т.к. нет между ними пересечений
non fast-forward - ветки пересекаются, поэтому сляние нужно править вручную

6. Когда может возникнуть конфликт слияния веток, как  его разрешить?
когда в нескольких ветках произошли новые коммиты после разделения. Решается слиянием и правкой вручную

7. Можно ли удалить ветку? Как? 
git branch -d <name>
